# :iphone: AcaR (NC1)
### Reflection App for Apple Developer Academy Student

Apple Developer Academy 여정을 위한 회고 앱 Acar은 </br>
회고 가이드를 제시해주어 의미있게 회고 할 수 있도록 돕고</br>
Challenge 별로 회고를 편하게 기록할 수 있습니다 :)

<br></br>
## :pushpin: Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform
